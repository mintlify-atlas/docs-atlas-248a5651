## What is Color Me?

Color Me is a browser extension that puts a vibrant, interactive color wheel right in your browser toolbar. Pick any color and your browser's frame, toolbar, and UI chrome instantly update to match — in real time.

Whether you want a calming blue for focus mode, a warm amber for evening browsing, or a bold red to match your mood, Color Me makes it effortless.

## How it works

Color Me uses the **WebExtensions Theme API** (`browser.theme.update()`) to dynamically update your browser's visual chrome. When you select a color on the wheel:

1. The hex value is captured from the `reinvented-color-wheel` component
2. The current mode (day or night) determines which theme profile to update
3. `browser.theme.update()` is called with the new frame color
4. The color is persisted to browser storage for the next session

<Note>
  Color Me primarily targets Firefox, where the Theme API has the broadest support. Chrome and Edge support is available but the Theme API has more limited scope on those platforms.
</Note>
