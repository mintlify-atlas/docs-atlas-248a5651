title: 'Quickstart'
description: 'Start customizing your browser theme in under 5 minutes'
---

<Note>
  Color Me is primarily tested on Firefox. The browser frame color customization feature relies on the Firefox `theme` API and will not work in other browsers.
</Note>

<Steps>
  <Step title="Build the extension">
    Clone the repository and build for your target browser:

    ```bash
    git clone https://github.com/ehcaw/colorme.git
    cd colorme
    yarn install
    yarn build:firefox
    ```

    See [Installation](/installation) for full build instructions and other browser targets.
  </Step>

  <Step title="Load it in Firefox">
    1. Navigate to `about:debugging` in Firefox.
    2. Click **Load Temporary Add-on**.
    3. Select any file inside the `colorme/dev/firefox` directory.

    Color Me will appear in your browser toolbar.
  </Step>

  <Step title="Open the popup">
    Click the **Color Me** icon in your browser toolbar to open the popup.

    The popup contains:
    - An interactive **color wheel** for selecting any color
    - A **hex value display** showing the currently selected color code
    - A **day/night mode toggle** checkbox
    - A **save button** (`+`) to add the current color to your palette
    - A **saved colors palette** showing your previously saved colors as clickable swatches
  </Step>

  <Step title="Pick a color">
    Click and drag on the color wheel to select a hue and saturation. The **hex value** below the wheel updates in real time to reflect your selection.

    <Tip>
      You can use the hex value displayed below the wheel to confirm the exact color before saving. It shows the live color code as you drag across the wheel.
    </Tip>
  </Step>

  <Step title="Toggle day or night mode">
    Use the **day/night mode toggle** checkbox to switch between your day and night theme colors. Color Me stores separate frame colors for each mode, so you can configure a light color for daytime and a dark color for nighttime.

    The browser frame updates immediately when you toggle between modes.
  </Step>

  <Step title="Save a color">
    Once you have chosen a color you want to keep, click the **`+` button** to save it to your palette. Saved colors appear as clickable color buttons in the saved colors palette at the bottom of the popup.

    Click any saved color swatch to instantly apply that color to the current mode.
  </Step>
</Steps>
